# A simple CLI program to track expenses and savings, with reports and basic analysis.

# Core Features, Add, View, Delete transactions. Each transaction has date: category(food,entertainment,bills,etc), description and amount.

# How to use, run the program by using python3 tracker.py, afterwards it will create a expenses.cbor file locally after initial run time and usage of the program

# CLI will give the options and what you can do with them
